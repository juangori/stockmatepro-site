Preguntas frecuentes

Necesito instalar algo?

No. StockMate Pro funciona directamente en tu navegador, como una pagina web. Tambien podes instalarlo como app en tu celular o computadora para acceso rapido, pero no es obligatorio.

Funciona en celular?

Si, esta disenado mobile-first. Podes usarlo desde cualquier celular, tablet o computadora. Incluso podes escanear codigos de barras con la camara del celular.

Puedo importar mis productos actuales?

Si. Podes importar tu catalogo completo desde un archivo CSV (Excel). El sistema reconoce codigo, descripcion, precio, costo, stock, color, talle y codigo de barras.

Como funciona la facturacion electronica?

StockMate Pro se conecta directamente con ARCA (ex AFIP) para generar facturas A, B y C con CAE valido. Solo necesitas tu certificado digital y punto de venta habilitado.

Mis datos estan seguros?

Si. Cada usuario tiene su propia base de datos aislada. Se realizan backups automaticos y la conexion es segura (HTTPS). Tus datos nunca se comparten con otros usuarios.

Puedo usarlo sin internet?

Si. StockMate Pro es una PWA (Progressive Web App) que funciona offline. Podes consultar tu inventario y registrar operaciones aunque no tengas conexion en ese momento.

Sirve para mi rubro?

StockMate Pro esta pensado para comercios minoristas: indumentaria, calzado, accesorios, bazares, ferreterias, librerias y cualquier negocio que maneje inventario fisico.
